# AI Projects Branch Strategy

이 저장소는 다음과 같은 브랜치 전략을 사용합니다.

## `main`
- **역할**: 최초 생성 브랜치 (Base branch)
- **특징**: 아무런 소스코드를 포함하지 않으며, 오직 이 브랜치 전략(정의서)만을 포함합니다.
- **용도**: 테스트 용 브랜치를 생성할 때 (필요한 상황이 발생할 경우) 이 `main` 브랜치에서 파생하여 생성합니다.

## `master`
- **역할**: 최종 배포 및 버전 관리 브랜치 (Production/Versioning branch)
- **용도**: 안정적인 버전 관리를 위해 사용되며, 버전 관리 브랜치를 만들 때 이 `master` 브랜치에서 파생하여 생성합니다.
