구현 및 검증

Overview

AI를 활용한 고도화된 프롬프트 작성부터 코드 리뷰, 리팩토링까지의 실전 과정을 안내합니다.

마이크로 프롬프트

작고 명확한 단위 기능(컴포넌트)을 빠르고 정확하게 생성하는 프롬프트 전략입니다.

🧩

컴포넌트 프롬프트 준비 중

버튼, 모달, 입력폼 등 개별 UI 요소 구현을 위한
프롬프트 템플릿이 업데이트 될 예정입니다.

아키텍쳐 프롬프트

전체 페이지 레이아웃과 복잡한 구조를 일관성 있게 구성하는 프롬프트 전략입니다.

🏗️

아키텍쳐 설계 준비 중

페이지 골격 구성, 컴포넌트 간 데이터 흐름 정의를 위한
매크로 레벨의 프롬프트 템플릿이 업데이트 됩니다.

리팩토링 프롬프트

코드를 더 깔끔하고, 재사용 가능하며, 효율적으로 개선하기 위한 프롬프트 전략입니다.

♻️

코드 개선 가이드 준비 중

AI를 활용해 레거시 코드를 리팩토링하고
클린 코드로 변환하는 팁이 곧 제공됩니다.

코드리뷰 프롬프트

작성된 코드의 잠재적 버그를 찾고, 성능을 검증하는 AI 리뷰 활용법입니다.

🔍

코드 검증 가이드 준비 중

접근성, 크로스브라우징, 에러 핸들링 등
AI 코드리뷰 체크리스트가 업데이트 될 예정입니다.
